Hardware notes:

1) lib holds all kicad symbols and footprints - All used symbols and footprints must be included in this location and be local to the project
2) Maintain clean libraries - Ask Ashton for help if needed
3) Resistors and Capacitors must be SMD - Standard size is 0603, only deviate when necessary
4) Specify values for caps/res as such - 0.1u (Cap) - 2k (res)
